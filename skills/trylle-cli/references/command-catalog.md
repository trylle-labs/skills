# trylle-cli CLI Map

Generated by `map_trylle_cli.py` using `--help` output.

- `auth`: Manage Trylle CLI credentials
  - `login`: Save a Trylle CLI Key and optional endpoints
  - `logout`: Remove saved Trylle CLI credentials
  - `status`: Show credential status
  - `token`: Print the active token source. Does not reveal token material unless --show is set

- `whoami`: Show the active Trylle identity
- `status`: Combined daily status: viewer, PRs, issues, and recent runs
- `repo`: Repositories
  - `list`: List repositories visible to the active account
  - `view`: Show repository metadata for OWNER/NAME or the current repo
  - `create`: Create a new repository for a user or organization
  - `clone`: Clone a repository using the clone URL returned by Trylle
  - `fork`: Fork a repository into the active account
  - `branches`: List branches for a repository
  - `edit`: Edit repository settings such as description, visibility, or default branch
  - `delete`: Delete a repository
  - `context`: Print the current repo, branch, current PR, viewer, and recent CI context

- `pr`: Pull requests
  - `list`: List pull requests for a repository
  - `view`: Show pull request details
  - `diff`: Show a pull request diff as structured JSON
  - `current`: Find the pull request associated with the current branch
  - `create`: Create a pull request from the current or specified branch
  - `edit`: Edit pull request title, body, state, or assignees
  - `checkout`: Check out the branch for a pull request
  - `comment`: Add a comment to a pull request
  - `update-branch`: Update a pull request branch from its base branch
  - `request-reviewers`: Request user or team reviewers on a pull request
  - `remove-reviewers`: Remove requested user or team reviewers from a pull request
  - `review`: Submit an approve, request-changes, or comment review
  - `dismiss-review`: Dismiss an existing pull request review
  - `review-comments`: List pull request review comments
  - `reply-review-comment`: Reply to a pull request review comment
  - `delete-review-comment`: Delete a pull request review comment
  - `changelog`: Show cached pull request changelog rows
  - `set-changelog`: Create or update one cached pull request changelog row
  - `status`: Show mergeability and check status for a pull request
  - `merge`: Merge a pull request
  - `close`: Close a pull request
  - `reopen`: Reopen a closed pull request

- `issue`: Issues
  - `list`: List issues for a repository
  - `view`: Show issue details
  - `create`: Create an issue
  - `edit`: Edit issue title, body, state, or assignees
  - `comment`: Add a comment to an issue
  - `comments`: List comments on an issue
  - `delete-comment`: Delete an issue comment by id
  - `reaction`: Add or remove a reaction on an issue body or issue comment
  - `label`: Add, remove, or replace issue labels
  - `close`: Close an issue
  - `reopen`: Reopen a closed issue
  - `timeline`: Show issue timeline events

- `stack`: Stacked pull request workflow
  - `create`: Create a branch intended for a pull request stack
  - `track`: Track parent metadata for a stacked pull request
  - `log`: Show the stack around the current branch or repository
  - `next`: Check out the next branch in the stack
  - `prev`: Check out the previous branch in the stack
  - `restack`: Rebase or restack the current branch series
  - `sync`: Sync child stack metadata for a pull request
  - `submit`: Submit the current stack branch as a pull request

- `org`: Organizations
  - `list`: List organizations visible to the active account
  - `view`: Show organization details
  - `members`: List organization members
  - `secret`: Manage organization actions secrets
    - `list`: List organization keys
    - `get`: Get an organization key by name
    - `set`: Set an organization key to a value
    - `delete`: Delete an organization key by name
  
  - `variable`: Manage organization actions variables
    - `list`: List organization keys
    - `get`: Get an organization key by name
    - `set`: Set an organization key to a value
    - `delete`: Delete an organization key by name
  

- `profile`: User profiles
  - `view`: Show a user profile
  - `follow`: Follow a user

- `ssh-key`: SSH keys
  - `list`: List SSH keys on the active account
  - `add`: Add an SSH public key
  - `delete`: Delete an SSH key by id

- `ci`: CI jobs and logs
  - `list`: List CI jobs for a repository
  - `view`: Show one CI run
  - `logs`: Print CI logs for a run
  - `watch`: Watch a CI run until it completes
  - `cancel`: Cancel a running CI job
  - `usage`: Show CI usage for a repository

- `actions`: Actions secrets and variables
  - `secret`: Manage repository actions secrets
    - `list`: List keys in the selected repository or organization scope
    - `get`: Get one key by name
    - `set`: Set a key to a value
    - `delete`: Delete one key by name
  
  - `variable`: Manage repository actions variables
    - `list`: List keys in the selected repository or organization scope
    - `get`: Get one key by name
    - `set`: Set a key to a value
    - `delete`: Delete one key by name
  

- `automation`: Automations
  - `list`: List automations for the active or selected account
  - `runs`: List recent runs for an automation
  - `enable`: Enable an automation
  - `disable`: Disable an automation

- `bot`: Bots
  - `list`: List bot accounts visible to the active account
  - `token`: Print or mint a token for a bot account

- `browse`: Print a dashboard URL for a Trylle resource
  - `repo`: Print the web URL for a repository
  - `pr`: Print the web URL for a pull request
  - `issue`: Print the web URL for an issue

- `api`: Raw Trylle API passthrough
- `about`: Show Trylle CLI copyright and license information
- `version`: Show version information, check for updates, or install the latest release
  - `check`: Check the package registry for a newer Trylle CLI release
  - `update`: Update Trylle CLI to the latest release

- `ask`: Ask a direct AI provider for quick help with Git and Trylle commands
- `agent`: Instruct a local agent CLI to perform work in the current context
- `skill`: Manage the Trylle CLI skill for local agent CLIs
  - `install`: Download and install the Trylle CLI skill
  - `update`: Download the latest Trylle CLI skill and replace the installed version
  - `uninstall`: Remove the Trylle CLI skill
  - `path`: Print the exact path to SKILL.md

- `explain`: Explain the changes in a commit, range, pull request, or current diff
- `draft`: Draft commit messages or branch names from the current diff
  - `branch`: Generate a branch name from the current diff
